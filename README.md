# Create-react-app and Electron together
> A sample of how to run this.

# How to run

1. Get the repo
```bash
git clone https://github.com/devraul/react-electron-sample.git
```

2. Install all depedencies
```bash
yarn install

#or

npm install
```

3. Run the application
```bash
yarn dev

#or

npm script dev
```
